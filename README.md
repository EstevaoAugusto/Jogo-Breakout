# Jogo-Breakout
Jogo Breakout desenvolvido puramente em HTML, CSS e Javascript. Projeto feito para disciplina 'Programação WEB'.

## Descrição do Jogo

Breakout é um clássico Jogo de quebrar blocos

## Estrutura do Projeto

- ```index.html```: Arquivo principal da aplicação, contêm a estrutura HTML do jogo.
- ```scripts/```: Contém os arquivos JavaScript responsáveis pela lógica do jogo.
- ```styles/```: Contém os arquivos CSS responsáveis pelo visual do jogo..
- ```sounds/```: Armazena os efeitos sonoros do jogo.

## Setup

Clone o projeto:

```bash
git clone git@github.com:EstevaoAugusto/Jogo-Breakout.git
```

Execute o arquivo 'index.html' diretamente, ou utilize a extensão Live Server do VS Code.
